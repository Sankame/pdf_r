# PDF回転スクリプト

このリポジトリには、PDFファイルを指定した角度で回転させるためのPythonスクリプト `rotate_pdf.py` が含まれています。

## 使用方法

1. このリポジトリをクローンします。
2. 必要なパッケージをインストールします。主に`PyPDF2`が必要です。
3. `rotate_pdf.py` を実行し、回転させたいPDFファイルと角度を指定して実行します。

### 依存パッケージのインストール
pip install PyPDF2

### スクリプトの実行例

    python rotate_pdf.py input.pdf output.pdf 90

上記コマンドは `input.pdf` を 90 度回転させて `output.pdf` として保存します。

## 注意事項

- 元ファイル (`input.pdf`) の内容が複雑な場合、回転後の結果に問題があるかもしれません。
- 出力ファイル名を指定することで元ファイルは変更されません。

### カスタマイズ

スクリプトの詳細やパラメータについては `rotate_pdf.py` を参照してください。必要に応じてカスタマイズが可能です。
